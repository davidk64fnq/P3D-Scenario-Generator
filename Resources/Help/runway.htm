Runway selection

There are two ways to select an airport. One is to click on the "Random" button, the other is to start typing in the "Search"" input field. In both cases the result will be displayed in the listbox to the right of the "Random button. There is no facility to browse a complete list of all available airports, the simulator provides this already.

The runways available in a new installation of P3D Scenario Generator corresponds to the default airports included with Prepar3D v5. These were generated using the freeware "MakeRunways" program (version 5.131) provided by Pete & John Dowson.

Search results show airport ICAO with the runway id in parentheses. P3D Scenario Generator doesn't display location information for the selected airport in order to preserve the element of surprise when a random runway is chosen.

Random button

Click on this to get a randomly selected runway. The chosen runway will be constrained by the Location Filter settings.

Search input field

Start typing a runway code and the runway listbox will update dynamically to show those runways that match the search string. The matching runways will NOT be constrained by the Location Filter settings.

Optional: Including addon runways

The runway data is derived from a file produced by the "MakeRunways" program named "runways.xml". To include add-on airports in the available list, download a copy of "MakeRunways" and generate a copy of "runways.xml" that reflects the airports installed on your PC.

This involves using a utility included with "MakeRunways" called "LorbySceneryExport", details are provided in the "MakeRwys README.pdf" file. Place the generated runways.xml file in the P3D Scenario Generator data directory. You can navigate to this location directly by opening the path: %APPDATA%\P3D Scenario Generator\ It will be used in preference to the default "runways.xml" list incorporated in the P3D Scenario Generator executable.

Note 1: Rather than double clicking MakeRwys.exe to execute it, use Windows Powershell from the directory containing Prepar3D.exe with the command ".\MakeRwys />0" This will ensure the really short runways are included rather than by default only greater 1500m.

Note 2: To include water runways use the command ".\MakeRwys /Water"

Note 3: Make sure all add-on scenery is activated before running MakeRwys!

Note 4: The onus is on the user to remember to create a new version of "runways.xml" after adding new airports to the simulator.

Note 5: Running MakeRwys on a default P3D v5.4 installation finds 56145 runways from 23514 airports. Of the 23514, 1038 airports have no runway data in the xml output file. The P3D Scenario Generator program ignores these airports with no runway data as the user interface and scenario generation logic are centred around knowledge of runways at an airport. If you have an airport that falls into this category and you want to see it included in your instance of the program you need to use ADE to add atleast one runway definition to the relevant BGL file.
