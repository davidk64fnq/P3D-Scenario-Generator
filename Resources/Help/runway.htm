Runway selection

The list of runways corresponds to the default airports included with Prepar3D v5. It was generated using the freeware "MakeRunways" program (version 5.131) provided by Pete & John Dowson.

The list is alphabetical showing airport ICAO and runway id in parentheses. P3D Scenario Generator doesn't display location information for the selected airport in order to preserve the element of surprise when a random runway is chosen.

Random button

Click on this to get a randomly selected runway. The chosen runway will be constrained by the Location Filter settings.

Runway combobox

Click the dropdown arrow and then drag to the desired airport. Alternatively typing a letter will take you to the first runway in the list beginning with that letter.

Search input field

Start typing a runway code and the runway combobox will update dynamically to show the first runway in the list that matches the search string. Assumes search string is starting with first letter of the runway.

Optional: Including addon runways

The runway data is derived from a file produced by the "MakeRunways" program named "runways.xml". To include add-on airports in the available list, download a copy of "MakeRunways" and generate a copy of "runways.xml" that reflects the airports installed on your PC.

This involves using a utility included with "MakeRunways" called "LorbySceneryExport", details are provided in the "MakeRwys README.pdf" file. Place the generated "runways.xml" file in the same directory as the P3D Scenario Generator program files and it will be used in preference to the default "runways.xml" list incorporated in the P3D Scenario Generator executable.

Note 1: Rather than double clicking MakeRwys.exe to execute it, use Windows Powershell from the directory containing Prepar3D.exe with the command ".\MakeRwys />0" This will ensure the really short runways are included rather than by default only greater 1500m.

Note 2: Make sure all add-on scenery is activated before running MakeRwys!

P3D Scenario Generator runs a check each time you generate a scenario to see whether a user generated "runways.xml" exists and if so that the last modified date of the file is more recent than that of "scenery.cfg" in the P3D Program Data folder. The first time it detects this a warning is displayed and then the "runways.xml" last modified date is set to now to stop the warning from repeating indefinitely.
