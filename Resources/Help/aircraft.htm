Aircraft selection

Start building a list of your favourite aircraft variants by using the "Add Aircraft" button.

Add Aircraft button

Click on this to select an aircraft variant to add to your list of favourites.

  1. To select an aircraft variant first navigate to the aircraft folder.
  2. For default aircraft this will be a folder within the "SimObjects/Airplanes" folder e.g. "SimObjects\Airplanes\Piper_J3Cub".
  3. For default helicopters it will be a folder within the "SimObjects/Rotorcraft" folder e.g. "SimObjects\Rotorcraft\Robinson_R22".
  4. Open the "texture.X" folder of the variant you require e.g. "SimObjects\Airplanes\Piper_J3Cub\Texture.1". The folder name must end in ".X" where X is any string of characters, choosing a "texture" folder won't work.
  5. Select the "thumbnail.jpg" file in that folder and click "Open".
  6. Confirm the chosen aircraft variant by hovering over the summary display field adjacent to the "Add Aircraft" button.

Note: Addon aircraft may be stored somewhere else on your PC. Just navigate to the appropriate "texture.X" folder and select the "thumbnail.jpg" file.
