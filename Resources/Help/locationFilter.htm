Location filter

The purpose of the location filters is to define the location in which a scenario is generated. For example you might wish to generate a scenario to fly using the Orbx True Earth Netherlands scenery. You would set up a location filter favourite that had a filter on country for the Netherlands. When editing location favourite filter values first select the location favourite from the Favourites combobox then make changes to country, state, and city using the relevant combo boxes.

Favourites combobox

Click the dropdown arrow and select an existing favourite.

To create (copy) a new favourite, select an existing favourite, type in the new favourite name and press the enter key. The new favourite country/state/city filters will be initialised to the values of the existing favourite.

To rename a favourite, select an existing favourite, type in the new favourite name and press the enter key, then select the old favourite name and press the delete key.

To delete a favourite, select an existing favourite, and press the delete key. You cannot delete the last remaining favourite in the list.

Country, State and City comboboxes

The following help is written in terms of country but applies equally to state and city.

To add a country to the currently selected favourite, click the country combobox dropdown arrow and and then drag to the desired country, then press the enter key. Alternatively typing a letter will take you to the first country in the list beginning with that letter.

To delete a country from the currently selected favourite, click the country combobox dropdown arrow and and then drag to the desired country, then press the delete key.

To delete ALL countries from the currently selected favourite, click the country combobox dropdown arrow and and then drag to the "None" entry (first in the list), then press the ENTER key.

Note: Changes are only made once the enter or delete key is pressed. Selecting a country without pressing enter or delete will have no effect on the currently selected favourite.

Random button

Click on this to get a randomly selected location favourite.

Favourite display field

Hover over the field adjacent to the random button to see a summary of the countries, states, and cities for the currently selected location favourite. The summary display remains for 5 seconds, if you select a different location favourite during that time you will need to wait till it clears before you can see the new summary.

Example

Scenario: You decide to create a location favourite for the Orbx True Earth products as you own all of them and for your next scenario you only want to fly in that scenery.

  1. Type the new location favourite name into the Favourites combobox and press enter e.g. "Orbx TE".
  2. Clear the country, state, and city filters by choosing "None" from the top of each combobox list and pressing enter.
  3. Select the country "Netherlands" from the country combobox and press enter.
  4. Select the country "United Kingdom" from the country combobox and press enter.
  5. Select the states "Washington", "California", "Florida", "Oregon" from the country comboboxes, one at a time and press enter.
  6. Confirm the location favourite has the correct filters by hovering over the summary display field adjacent to the random button.
