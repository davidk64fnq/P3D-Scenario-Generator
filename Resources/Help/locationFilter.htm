Location filter

The purpose of the location filters is to control the location in which a scenario is generated. For example you might wish to generate a scenario to fly using the Orbx True Earth Netherlands scenery. You would set up a location filter favourite that had a filter on country for the Netherlands.

Favourites combobox

Click the dropdown arrow and select an existing favourite.

To create (copy) a new favourite, select an existing favourite, type in the new favourite name and the enter key. The new favourite country/state/city filters will be initialised to the values of the existing favourite.

To rename a favourite, select an existing favourite, type in the new favourite name and press the enter key, then select the old favourite name and press the delete key.

To delete a favourite, select an existing favourite, and press the delete key. You cannot delete the last remaining favourite in the list.
